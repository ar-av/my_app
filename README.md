# Gym Tracker App

## Overview
This is a Flutter app that helps users track their gym workouts and progress.

## Features
- Track Push, Pull, and Leg workouts
- Store user details (name, age, height, weight)
- Weekly progress tracking
- Dark themed UI

## Tech Stack
- Flutter
- Dart
